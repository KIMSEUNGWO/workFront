1. 자료형

자바스크립트에서는 변수에 별도로 자료형을 지정해주지 않고, [리터럴]에 의해 자료형이 결정된다.
리터럴 : 변수에 대입되는 값

자료형을 확인 하는 방법
typeof(값 또는 변수명)

Primitive type 원시타입

하나의 값만 저장하는 자료형
  • string
  • number
  • boolean

    • 0, "" '' `` 빈문자열, NaN, undefined, null = false
    res= 0 ? 'true' : 'false'

    truthy : true로 인정할 수 있는 값 (falsy를 제외한 값)
  • undefined

    • 변수를 선언만하고 값을 할당하지않았을때의 초기값

  • null

    • 값이 없거나 유효하지 않는 값

    2. 데이터 형변환

    1)문자열과 숫자의 '+' 연산

    문자열 + 숫자 = 문자열
    숫자 + 숫자 + 문자열 = 문자열 숫자가 먼저 계산되고 뒤의 문자열과 합쳐진다

    2) 강제 형변환

    Number(값), parseInt(값), parseFloat(값)
    문자형, 논리형 -> 숫자형 변환

    String(값), 값.toString()
    숫자형, 논리형 -> 문자형 변환

    Boolean(값)
    문자형, 숫자형 -> 논리형 변환
